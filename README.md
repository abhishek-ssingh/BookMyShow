# BookMyShow
Design Book My Show application

Come up with application that can perform

- Booking tickets
- Search Movies, Meetups, Shows
- Search theatres
- Give ratings

# Requirements
## 1. Client 
- Search events based on Interests, names, ratings
- Search for available seats
- Select a format (movies)
- View Orders
- Nearby Events
- Share/Split tickets
- Payment
- Rating 

## 2. Event Organizers
- Add Events
  - Movies
  - Shows
  - Meetups
  - Workshops
- Available Seats
- Available formats

## 3. System
- Onboarding
- Notification
- Share/Split tickets
- Event Location
- Payments
